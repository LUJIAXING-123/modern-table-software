# 现代表格软件

一个功能完善的表格软件，支持用户自定义每一列的数据类型，UI简洁现代美观。

## 功能特点

### 📋 核心功能
- **自定义数据类型**：支持文本、整数、布尔值三种数据类型
- **动态增删行列**：可灵活添加和删除表格的行和列
- **类型验证**：自动验证和转换数据，确保数据类型一致性
- **直观编辑**：双击单元格即可编辑数据
- **数据持久化**：支持保存和加载数据到JSON文件

### 🎨 UI设计
- **现代简洁**：采用TTK主题，界面清爽美观
- **响应式布局**：支持水平和垂直滚动
- **直观操作**：工具栏按钮一目了然
- **类型配置面板**：方便的列类型配置区域

### 💡 数据类型支持
| 数据类型 | 说明 | 输入方式 |
|---------|------|---------|
| 文本 | 普通文本数据 | 文本输入框 |
| 整数 | 数值类型，自动转换 | 数值微调框 |
| 布尔值 | 二选一选项 | 单选按钮组 |

## 技术栈

- **编程语言**：Python 3.14
- **GUI框架**：wxPython 4.2.1
- **数据存储**：JSON格式

## 安装和运行

### 系统要求
- Python 3.7 或更高版本
- 支持 wxPython 的系统（Windows、macOS、Linux）

## 使用指南
1.下载release版本的可执行文件
2.运行可执行文件
3.开始使用！

### 基本操作

1. **添加列**：点击工具栏的「添加列」按钮
2. **添加行**：点击工具栏的「添加行」按钮
3. **删除列**：选择要删除的列，点击「删除列」按钮
4. **删除行**：选择要删除的行，点击「删除行」按钮
5. **编辑单元格**：双击要编辑的单元格
6. **配置列类型**：在下方的「列类型配置」区域选择列的数据类型
7. **保存数据**：点击「保存数据」按钮，选择保存位置
8. **加载数据**：点击「加载数据」按钮，选择要加载的JSON文件
9. **导入xlsx文件**：点击「导入xlsx文件」按钮，选择要导入的xlsx文件

## 功能演示

### 1. 创建表格
- 软件启动时自动创建一个默认的空表格
- 可根据需要添加更多行列

### 2. 配置数据类型
- 在下方的类型配置区域，为每一列选择合适的数据类型
- 更改类型后，现有数据会自动进行验证和转换

### 3. 编辑数据
- 双击单元格进入编辑模式
- 根据列类型，显示不同的编辑控件：
  - 文本：普通输入框
  - 整数：数值微调框
  - 布尔值：单选按钮组

### 4. 保存和加载
- 支持将表格数据保存为JSON文件
- 可随时加载之前保存的数据文件

## 开发说明

### 代码结构

- `TableApp` 类：主应用类，管理整个表格软件
- `create_toolbar()`：创建工具栏
- `create_table_frame()`：创建表格显示区域
- `create_type_config_frame()`：创建类型配置区域
- `add_column()/add_row()`：添加行列
- `delete_column()/delete_row()`：删除行列
- `update_table()`：更新表格显示
- `validate_and_convert()`：数据类型验证和转换
